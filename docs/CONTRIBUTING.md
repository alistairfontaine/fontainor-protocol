# Fontainor v0.5.0 Production Development Guidelines

1. **Protocol Sovereignty:** Every feature must respect the "Asset-First" philosophy. No centralized tracking trackers or telemetry.
2. **Serverless Autonomy:** All mutations to the global registry ledger must remain client-signed and wallet-funded to eliminate hosting overhead.
3. **Collector's Equity:** The core logic of resale royalties and automatic 98/2 treasury splits must remain immutable within the verification engines.
4. **Lean Stack:** Keep the runtime footprint minimal. If a feature doesn't directly contribute to decentralized distribution, it is out of scope.
